# Brag Plan: The Coder Bros

## Creative Direction
- **Tone:** app-store (clean, professional, feature-forward)
- **Angle:** A modern coding education platform that's hands-on from the start — interactive courses, live code labs, and practical cheatsheets for 5 languages.
- **Hook:** Full-screen "The Coder Bros" + tagline "Master Modern Development" with the signature blue-purple gradient
- **Outro:** "Start Learning at thecoderbros.vercel.app"

## Storyboard (20s)

### Scene 1 — Hero Reveal (0-3.5s)
- Dark background with animated purple gradient orbs
- Swoosh line draws horizontally across center
- "The Coder Bros" slides up from below with power4 ease (Apple-style reveal)
- Glow burst behind text fades in then out
- "Master Modern Development" tagline slides up in uppercase with letter-spacing

### Scene 2 — Languages (3.5-9s)
- "Learn Five Languages" header with animated gradient underline
- 5 language cards (Python, JavaScript, Java, C#, C++) enter staggered with back.out bounce + rotationX
- Each card has an SVG icon (no emojis) and colored top accent bar via ::before

### Scene 3 — Code Lab (9-14.5s)
- Code window slides up with scale shadow effect
- "● READY" status light blinks in
- Python code lines fade in sequentially with slide
- Terminal output appears in a green-tinted box with blinking cursor
- Tagline: "Interactive labs. Real code. Real results."

### Scene 4 — Features (14.5-17.5s)
- Two premium cards slide in from left/right with rotationY
- "Courses" card with code window SVG icon, 3 feature lines
- "Cheatsheets" card with database/file SVG icon, 3 feature lines

### Scene 5 — Outro (17.5-20s)
- Logo scales in with back.out bounce + burst glow
- Tagline fades up in uppercase
- "Start Learning" CTA button with gradient
- URL fades in subtly

## Audio
- **Music:** happy-beats-business-moves-vol-1-by-ende-dot-app (full upbeat track)
- **Music role:** warm bed, professional energy
- **SFX:** light interface drops for card entrances, soft bell for logo reveal, keyboard clicks for code scene
- **Audio-reactive treatment:** subtle; music RMS makes the gradient orbs breathe
