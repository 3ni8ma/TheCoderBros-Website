# The Coder Bros — Master Modern Development

Interactive coding courses, cheatsheets, and labs for Python, JavaScript, Java, C#, and C++.

**Live site:** [thecoderbros.vercel.app](https://thecoderbros.vercel.app) (Vercel — full Next.js app with API routes)  
**GitHub Pages:** [3ni8ma.github.io/TheCoderBros-Website](https://3ni8ma.github.io/TheCoderBros-Website/) (redirects to main site)

Built with Next.js 16, React 19, Tailwind CSS 4, Framer Motion, Monaco Editor.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```
